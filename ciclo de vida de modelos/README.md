# Ciclo de Vida de Modelos

Repositório principal da disciplina para organizar exercícios práticos sobre o ciclo de vida de modelos de Machine Learning.

## Objetivo do repositório

Centralizar os materiais práticos da disciplina em um único lugar, com foco em:
- experimentação de modelos;
- preparação e qualidade de dados;
- avaliação de desempenho;
- rastreabilidade e reprodutibilidade;
- boas práticas de desenvolvimento em projetos de ML.

## Organização do repositório

A ideia é manter uma estrutura por aulas, tópicos ou módulos:

```text
ciclo de vida de modelos/
├── Aula 1/
├── Aula 2/
├── Aula 3/
├── data/
├── docs/
└── README.md
```

## Como usar este repositório

1. Entre na pasta da aula ou tema desejado.
2. Leia o material específico daquele exercício.
3. Execute notebooks/scripts conforme as instruções locais.
4. Mantenha dados e artefatos organizados para facilitar reuso.

## Tecnologias comuns

- Python
- Jupyter Notebook
- pandas, numpy, matplotlib, seaborn
- scikit-learn
- MLflow (quando aplicável)
